## Если проблемы остались:

```bash
sqlite3 ~/.carrental/app.db "SELECT COUNT(*) FROM cars;"
```

Должно быть 50.

### Полная очистка и пересборка:

```bash
cd /home/vladius/Desktop/apps/CarRENTAL

# Удаляем базу данных
rm -f ~/.carrental/app.db

# Удаляем старую сборку
rm -rf build
mkdir build
cd build

# Собираем заново
cmake ..
cmake --build . -j$(nproc)

# Запускаем
./bin/CarRental
```
