# Инструкция по запу Pocketму CarRental

## Быстрый запуск (если уже собрано)

Если проект уже был собран ранее, просто запустите:

```bash
cd /home/vladius/Desktop/apps/CarRENTAL/build
./bin/CarRental
```

## Полная пересборка (после изменений)

После внесенных изменений нужно пересобрать проект:

### Вариант 1: Полная пересборка

```bash
cd /home/vladius/Desktop/apps/CarRENTAL
rm -rf build          # Удаляем старую сборку
mkdir build
cd build
cmake ..
cmake --build . -j$(nproc)
```

### Вариант 2: Быстрая пересборка (если build уже существует)

```bash
cd /home/vladius/Desktop/apps/CarRENTAL/build
cmake ..
cmake --build . -j$(nproc)
```

## Запуск приложения

```bash
cd /home/vladius/Desktop/apps/CarRENTAL/build
./bin/CarRental
```

Или можно создать алиас/ярлык для удобства.

## Что происходит при первом запуске

1. Приложение создаст базу данных SQLite в `~/.carrental/app.db`
2. Автоматически добавятся 50 автомобилей в базу данных
3. Откроется окно входа/регистрации

## Требования

- CMake >= 3.21 ✅ (установлен: 3.28.3)
- Qt6 или Qt5 ✅ (установлен: Qt 6.4.2)
- C++17 компилятор ✅

## Устранение проблем

### Если не компилируется из-за отсутствия Qt:

```bash
sudo apt update
sudo apt install -y qt6-base-dev qt6-base-dev-tools libqt6sql6-sqlite
```

### Если ошибка "Qt not found":

Убедитесь, что Qt установлен:

```bash
qmake --version
```

### Если нужно использовать Qt5 вместо Qt6:

Установите Qt5:

```bash
sudo apt install -y qtbase5-dev qtbase5-dev-tools libqt5sql5-sqlite
```

## Использование приложения

1. **Вход/Регистрация**: При первом запуске введите имя, телефон и email, затем нажмите "Зарегистрироваться"
2. **Просмотр автомобилей**: Переключитесь на вкладку "Автомобили"
3. **Поиск**: Используйте поле поиска для фильтрации автомобилей
4. **Выбор валюты**: Измените валюту (USD/EUR/BYN) в правом верхнем углу
5. **Аренда**: Нажмите "Арендовать" на карточке автомобиля
6. **Мои аренды**: Перейдите на вкладку "Мои аренды" для просмотра ваших бронирований
